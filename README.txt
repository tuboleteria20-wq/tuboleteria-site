
TuBoletería - Sitio estático (HTML/CSS/JS)
Archivos en esta carpeta:
- index.html
- styles.css
- script.js
- assets/ (logo y placeholders)
Cómo usar:
1) Reemplaza imágenes en assets/ por tus fotos (mismos nombres o ajustar index.html).
2) Edita el número de WhatsApp y el correo en index.html y script.js si cambias de número.
3) Opciones para publicar:
   - GitHub Pages (gratuito para sitios estáticos)
   - Netlify (arrastrar y soltar ZIP o conectar a GitHub)
   - Host con cPanel/FTP (sube los archivos a public_html)
Ver las instrucciones detalladas en el archivo 'INSTRUCCIONES_DE_PUBLICACION.txt' (puede estar fuera de este ZIP).
